Welcome to my Open Source Tool Dashboard
Cyber Security Tools are important, and can be even considered part of your knowledge base. The reason why the website is being made so I can categorize tools I have come across and make a neat looking hub to store them.


WireShark

Wireshark is an open-source software application that is one of the most popular network protocol analyzers. Many people use it to capture and analyze data across a computer network. It facilitates detailed analyses of packets, exposing network protocol layers so users can effectively stare into events inside the network and manage flows, perform protocol investigations, probe faults, and explore their network and security domains. It is considered one of the most significant essential tools to have at your disposal. Github Home Page Installation Guide

Nmap

Network Mapper (known as Nmap) is a freely available tool for network exploration and security assessment purposes. This software works by examining collapsibleButton through the transmission of packets and the responses they receive. It offers insights into hosts' details and information about services and accessible network ports. The versatility of Nmap allows it to perform functions like discovering vulnerabilities in systems, creating inventories of collapsibleButton, monitoring the availability of services, and detecting any unauthorized devices connected to the network. It is one of the most basic tools to utilize and one of the most essential. Home Page Installation Guide

RustScan

A rapid port scanner that can scan all 65k ports in 3 seconds. It boasts full scripting engine support that can pipe output into Nmap. It uses adaptive learning that will fine-tune itself to match your OS. It also allows automatic piping into Nmap for every (ethical) hacker's desire. Github Home Page Installation Guide

ns-3

ns-3 is a discrete-event network simulator, that is targeted primarily for research and educational use. ns-3 software infrastructure encourages the development of simulation models which are sufficiently realistic to allow ns-3 to be used as a realtime network emulator, interconnected with the real world and which allows many existing real-world protocol implementations to be reused within ns-3. Very helpful if setting up any collapsibleButton and need to check its validaility. Home Page Versions for Installation

Suricata

Suricata is a network analysis and threat detection software used by most private and public organizations, and embedded by major vendors to protect their assets. Suricata boasts an impressive line up of features that include Lua scripting, IDS/IPS, High Performance (I would hope so), Logging HTTP requests, logging and storing TLS certificates, and extracting files from flows and store them to disk. Very widely used, and loved, SIEMs are the backbone of our security so it is imparative to have one. Home Page Installation Page

OSSIM

OSSIM is a C++ written, geospatial image processing library used by many entities throughout the solar system. It has been in active development since the last millenium. It supports customizing your build by using environment variables and by directly editing the cmake configuration script. Github Page Installation Guide

Alien Vault

Open Threat Exchange aka Alien Vault allows everyone to openly collaborate to share the latest information about emerging threats , attack methods, and malicious actors. Home Page

TheHive

TheHive is a scalable 3-in-1 open source and free Security Incident Response Platform that is designed to make life easier for SOCs, and all information security practitioner. It can be synchronized with multiple instances of MISP to start investigations. Github Page Installation Guide

WebScarab

WebScarab's primary purpose is to expose the workings of an HTTP(S)-based application. It is a framework for analyzing applications that communicate using the HTTP and HTTPS protocols. It is highly suggested that any user who uses this has a high knowledge of the HTTP protocol. Github Page Installation
