Networking tools

WireShark

Wireshark is an open-source software application that is one of the most popular network protocol analyzers. Many people use it to capture and analyze data across a computer network. It facilitates detailed analyses of packets, exposing network protocol layers so users can effectively stare into events inside the network and manage flows, perform protocol investigations, probe faults, and explore their network and security domains. It is considered one of the most significant essential tools to have at your disposal. Github Home Page Installation Guide

Nmap

Network Mapper (known as Nmap) is a freely available tool for network exploration and security assessment purposes. This software works by examining networks through the transmission of packets and the responses they receive. It offers insights into hosts' details and information about services and accessible network ports. The versatility of Nmap allows it to perform functions like discovering vulnerabilities in systems, creating inventories of networks, monitoring the availability of services, and detecting any unauthorized devices connected to the network. It is one of the most basic tools to utilize and one of the most essential. Home Page Installation Guide

RustScan

A rapid port scanner that can scan all 65k ports in 3 seconds. It boasts full scripting engine support that can pipe output into Nmap. It uses adaptive learning that will fine-tune itself to match your OS. It also allows automatic piping into Nmap for every (ethical) hacker's desire. Github Home Page Installation Guide

ns-3

ns-3 is a discrete-event network simulator, that is targeted primarily for research and educational use. ns-3 software infrastructure encourages the development of simulation models which are sufficiently realistic to allow ns-3 to be used as a realtime network emulator, interconnected with the real world and which allows many existing real-world protocol implementations to be reused within ns-3. Very helpful if setting up any networks and need to check its validaility. Home Page Versions for Installation
Security Monitoring, Intrusion detection/prevention

Suricata

Suricata is a network analysis and threat detection software used by most private and public organizations, and embedded by major vendors to protect their assets. Suricata boasts an impressive line up of features that include Lua scripting, IDS/IPS, High Performance (I would hope so), Logging HTTP requests, logging and storing TLS certificates, and extracting files from flows and store them to disk.
