RustScan

A rapid port scanner that can scan all 65k ports in 3 seconds. It boasts full scripting engine support that can pipe output into Nmap. It uses adaptive learning that will fine-tune itself to match your OS. It also allows automatic piping into Nmap for every (ethical) hacker's desire. Github Home Page Installation Guide
